# QUAL A DIFERENÇA ENTRE BIBLIOTECA E FRAMEWORK?

## **Biblioteca**
É um conjunto de códigos prontos que você pode usar no seu projeto para resolver tarefas específicas. Você continua no controle do fluxo do programa e chama a biblioteca quando precisar.
Exemplo: você usa uma função da biblioteca para fazer requisições HTTP ou manipular datas.

## **Framework**
É uma estrutura mais completa que define como o seu projeto deve ser organizado. Ele controla o fluxo da aplicação e você encaixa seu código dentro dele.
Exemplo: no Flutter ou no Spring Boot, você segue regras e padrões já definidos para construir a aplicação.

## **Diferença principal**
A diferença está no controle:

* Na **biblioteca**, você chama o código quando quiser.
* No **framework**, ele chama o seu código (isso é conhecido como “inversão de controle”).
